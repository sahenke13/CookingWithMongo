# CookingWithMongo

created by Steven Henke

This is an application which scraps a cooking website for recipes.

Utilizing Mongo's Database, express, handlebars, cheerio and axios this application allows the user to scrape food networks all-articles section, which will display with a Title which is linked to the article, as well as a summary. The user can then save articles for later, and make comments to articles which have been saved.




This application is deployed at: https://guarded-wave-97750.herokuapp.com/
